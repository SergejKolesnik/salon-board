# cosmo-schedule

Розклад косметологічного кабінету — однофайловий веб-застосунок на FastAPI.

## Можливості

- **Сітка майстрів** — всі спеціалісти бачать навантаження один одного в реальному часі
- **Рецепція** може створювати, редагувати та видаляти записи
- **Перевірка конфліктів** — система не дозволить записати двох клієнтів на той самий час до одного майстра
- **Мобільний вигляд** — вкладки по майстрах, адаптований під телефон
- **Клік на слот** — форма одразу з заповненими часом і майстром
- **SQLite** — база створюється автоматично, без налаштувань

## Технології

- **Backend:** Python 3.10+, FastAPI, SQLite (вбудований)
- **Frontend:** Vanilla JS, HTML/CSS — вбудований в `main.py`, без збірки
- **Одна залежність:** `fastapi` + `uvicorn`

## Запуск

```bash
# 1. Клонувати
git clone https://github.com/YOUR_USERNAME/cosmo-schedule.git
cd cosmo-schedule

# 2. Встановити залежності
pip install -r requirements.txt

# 3. Запустити
uvicorn main:app --reload

# 4. Відкрити у браузері
# http://localhost:8000
```

База даних `cosmo.db` створюється автоматично при першому запуску з демо-даними (4 майстри, записи на сьогодні).

## API

| Метод | Endpoint | Опис |
|-------|----------|------|
| GET | `/api/masters` | Список майстрів |
| POST | `/api/masters` | Додати майстра |
| GET | `/api/appointments?date=YYYY-MM-DD` | Записи на дату |
| POST | `/api/appointments` | Створити запис |
| PUT | `/api/appointments/{id}` | Оновити запис |
| DELETE | `/api/appointments/{id}` | Видалити запис |
| GET | `/api/breaks?date=YYYY-MM-DD` | Перерви на дату |
